Suas escolhas definem o seu destino. 
O maior desafio é saber qual escolha é a correta. Isso se houver alguma.

1 - Extraia os arquivos de Alcatraz.rar para qualquer pasta.

2 - Execute o arquivo Alcatraz.exe
 
Como jogar:

O jogo consiste em decisões. Você terá a oportunidade de escolher entre ações diferentes que definem o rumo da história.
Pense bem antes de responder, pois CADA uma das respostas definirá seu rumo.
Para escolher a ação pressione o número correspondente.
Quando houver linhas com apenas '...' pressione qualquer tecla para continuar.
Fechar o executável fará com que o estado do jogo salvo se perca.
Na tela de menu pressione 'M' para rever as instruções.
